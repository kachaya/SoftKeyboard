# SoftKeyboard
## 概要
Android用のKeyboadViewを使わない最小限のソフトキーボードです。
- PCの日本語キーボードに似たQWERTYキー配列
- キー配置はlayoutファイルにTextViewとImageViewを配置
- キーリピートのタイミングはシステム設定に従う
